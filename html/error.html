No Database Found:

1. Check if the file is located on a (local/web) server and if the server is running, as well as if mySQL is active.
If not:
- Download and install XAMPP from Download
- Then start the Apache server and mySQL, and place the folder in the directory: "C:\xampp\htdocs"
- Then open http://localhost/phpmyadmin/
- Create a new database with the name: projekt_c37592b with utf8_unicode_ci
- Select the created database on the left and choose the "Import" tab at the top
- First import the file: database.sql.gz and confirm with OK
- Repeat the last two steps with the file: SeilbahnDaten.sql.gz
- Then open in the browser: Homepage
- You should not be redirected here
